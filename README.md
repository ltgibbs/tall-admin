## Installation

### Clone Repo

```bash
git clone https://github.com/manusiakemos/tall-admin.git
```

### Run Command

```bash
cd tall-admin

composer i

cp .env.example .env
#configure .env variables

php artisan key:generate

npm i

npm run prod 
#or npm run dev for development

php artisan migrate

php artisan storage:link

php artisan db:seed
```
